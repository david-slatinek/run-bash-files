Run bash files - documentation

Information

Name: Run bash files

Purpose: Run all sh files found at path

If only specific files are required to run, check -r flag

Synopsis

Synopsis
Usage Results
bash script.sh OR bash script.sh -h Prints help
bash script.sh [path] {flags} Run sh files found at [path] with {flags}. Not valid for -h or -f flags
chmod +x script.sh => ./script.sh [path] {flags} Same as above

Flags

Flags
Flag ID Flag symbol Meaning Description
1 -h Help Print help
2 -r [regex] Regex Run only files whose name matches with [regex]
3 -R Recursive mode Enable recursive mode
4 -q Quiet mode Redirects file output to /dev/null
5 -p Print Before running a file, print path and filename
6 -f Print without running the file Print path and filename without running the file
7 -s Statistics Print statistics
8 -e Error Print errors when running files
